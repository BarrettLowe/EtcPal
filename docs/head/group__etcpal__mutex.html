Detailed Description

A mutual-exclusion (mutex) object.

Sometimes also referred to as a critical section. Only one thread can take the mutex at a time using etcpal_mutex_take(). Where possible on real-time platforms, the mutex provides priority inheritance.

etcpal_mutex implementations use the following constructs under the hood on various platforms:

On: etcpal_mutex uses:
FreeRTOS Mutexes
Linux pthread_mutex
macOS pthread_mutex
MQX Lightweight Semaphores
Windows Critical Section objects
