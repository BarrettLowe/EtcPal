Detailed Description

(Not for direct usage) A memory pool description structure.

Do not declare or use this structure directly; instead, use ETCPAL_MEMPOOL_DECLARE(), ETCPAL_MEMPOOL_DEFINE(), ETCPAL_MEMPOOL_DEFINE_ARRAY(), and the etcpal_mempool_* macros to interact with it.
